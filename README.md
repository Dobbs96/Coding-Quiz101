# Code-Quiz

## Story

```
AS A coding boot camp student
I WANT to take a timed quiz on JavaScript fundamentals that stores high scores
SO THAT I can gauge my progress compared to my peers
```

## Accepted Criteria

- [x] Start button
- [x] Answer question
  - [x] Correct
    - [x] Add Score
  - [x] Wrong
    - [x] Subtracted time
- [x] Game over when
  - [x] All qutions answered
  - [x] Time runs out
- [] Save score
- [x] No Errors
- [x] Posted
- [x] Live URL
